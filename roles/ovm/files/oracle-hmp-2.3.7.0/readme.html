Oracle Hardware Management Pack 2.3.7.0 contains Management Agents, SNMP Plugins, and CLI Tools for your Oracle Sun x86 servers, SPARC servers, and blade modules.

These cross platform Oracle Hardware Management Pack components enable you to:

  • Use a management agent at the operating system level to enable in-band monitoring of your Oracle server hardware over Simple Network Management Protocol (SNMP). You can use this information to integrate your servers into your data centre management infrastructure.
  • Configure UEFI and legacy BIOS, RAID volumes and Oracle Integrated Lights Out Manager (ILOM) service processors on your servers.
  • View hardware configuration and status information, and upgrade your server components' firmware.
  • Configure an trap proxy that forwards SNMP traps from your Oracle ILOM service processor over the Host-to-ILOM Interconnect.
  • Configure zoning on supported servers running Oracle Solaris.
  • Use IPMItool to access server service processors via the IPMI protocol and perform management tasks.
  • Use the HMP watchdog agent to perform pre-set actions if either the host or Oracle ILOM become unresponsive on a Linux system.

For more information on Oracle Hardware Management Pack, including the support matrix showing which hardware is supported by the different components, please visit: http://www.oracle.com/goto/ohmp.
